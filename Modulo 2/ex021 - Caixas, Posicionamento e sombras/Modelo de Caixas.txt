Caixas são o local que ficam os elementos de HTML

em <h1>Olá, mundo!</h1>, o Olá, mundo! está dento da caixa h1.

As medidas de box são:
height = altura, por padrão é a altura minima para caber o elemento
width = largura, por padrão é a altura minima para caber o elemento
border = uma linha que circula nossa caixa(box)
padding = espaço interno entre o elemento e o border
margin = espaço externo da borda, um espaço entre as Caixas
outline = uma
